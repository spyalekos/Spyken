# 🎵🎬 Spyken — Έγγραφο σε Ομιλία (MP3) & Βίντεο (MP4)

**by [spyalekos](https://github.com/spyalekos)**

---

## Τι κάνει η εφαρμογή

Το **Spyken** μετατρέπει αρχεία `.docx` και `.pdf` σε αρχεία ήχου `.mp3` ή δημιουργεί αυτόματα παρουσιάσεις βίντεο `.mp4` χρησιμοποιώντας τεχνολογία σύνθεσης ομιλίας (Text-to-Speech) μέσω της υπηρεσίας Microsoft Edge TTS.

### Κύρια χαρακτηριστικά

- 📄 **Υποστηριζόμενες μορφές**: `.docx` (Word) και `.pdf`
- 🔊 **Εναλλαγή φωνών**: Κάθε παράγραφος διαβάζεται εναλλάξ από ανδρική και γυναικεία φωνή, για πιο φυσική ακρόαση.
- 🇬🇷 **Ελληνικές φωνές**: `el-GR-NestorasNeural` και `el-GR-AthinaNeural`.
- 🇬🇧 **Αγγλικές φωνές**: Το αγγλικό κείμενο ανιχνεύεται αυτόματα και διαβάζεται με σωστή προφορά (`en-GB-RyanNeural`/`en-GB-SoniaNeural`).
- 🎬 **Δημιουργία Βίντεο (MP4)**: Εκτός από σκέτο ήχο, μπορεί να παράγει βίντεο όπου το κείμενο της τρέχουσας παραγράφου προβάλλεται στην οθόνη συγχρονισμένα με τη φωνή που ακούγεται (υποστηρίζει εξαγωγή σελίδων PDF ως εικόνες ή δυναμική αναδίπλωση κειμένου για DOCX).
- 📦 **Πολλαπλά αρχεία**: Μπορείτε να επιλέξετε και να μετατρέψετε πολλά αρχεία (batch processing).
- 📊 **Γραμμή προόδου**: Εμφανίζει την πρόοδο (ήχου και κατασκευής βίντεο) σε πραγματικό χρόνο.
- 🌙 **Dark mode**: Ενσωματωμένο, μοντέρνο και φιλικό προς το χρήστη περιβάλλον.

---

## Πώς χρησιμοποιείται

1. Ανοίξτε το εκτελέσιμο `Spyken.exe`.
2. Πατήστε **«Επιλογή Αρχείων»** και προσθέστε τα `.docx` ή `.pdf` σας.
3. Επιλέξτε τι θέλετε να κάνετε:
   - Πατήστε **«Μετατροπή σε MP3»** για δημιουργία αρχείων ήχου.
   - Ή πατήστε **«Μετατροπή σε MP4»** για δημιουργία βίντεο-παρουσίασης.
4. Τα παραγόμενα αρχεία αποθηκεύονται στον ίδιο φάκελο με τα αρχικά έγγραφα.

| GUI | [Flet](https://flet.dev) |

| PDF | PyMuPDF (fitz) |
| DOCX | python-docx |
| Βίντεο | MoviePy, Pillow (PIL), NumPy |
| Build | PyInstaller `--onefile --windowed` |
