# 🎵 Spyken — Έγγραφο σε Ομιλία (MP3)

**by [spyalekos](https://github.com/spyalekos)**

---

## Τι κάνει η εφαρμογή

Το **Spyken** μετατρέπει αρχεία `.docx` και `.pdf` σε αρχεία ήχου `.mp3` χρησιμοποιώντας τεχνολογία σύνθεσης ομιλίας (Text-to-Speech) μέσω της υπηρεσίας Microsoft Edge TTS.

### Κύρια χαρακτηριστικά

- 📄 **Υποστηριζόμενες μορφές**: `.docx` (Word) και `.pdf`
- 🔊 **Εναλλαγή φωνών**: Κάθε παράγραφος διαβάζεται εναλλάξ από ανδρική και γυναικεία φωνή, για πιο φυσική ακρόαση
- 🇬🇷 **Ελληνικές φωνές**: `el-GR-NestorasNeural` (ανδρική) και `el-GR-AthinaNeural` (γυναικεία)
- 🇬🇧 **Αγγλικές φωνές**: Το αγγλικό κείμενο ανιχνεύεται αυτόματα και διαβάζεται από `en-GB-RyanNeural` (ανδρική) και `en-GB-SoniaNeural` (γυναικεία) για σωστή προφορά
- 📦 **Πολλαπλά αρχεία**: Μπορείτε να επιλέξετε και να μετατρέψετε πολλά αρχεία ταυτόχρονα
- 📊 **Γραμμή προόδου**: Εμφανίζει την πρόοδο της μετατροπής σε πραγματικό χρόνο
- 🌙 **Dark mode** ενσωματωμένο

---

## Πώς χρησιμοποιείται

1. Ανοίξτε το `Spyken.exe`
2. Πατήστε **«Επιλογή Αρχείων»** και επιλέξτε ένα ή περισσότερα `.docx`/`.pdf` αρχεία
3. Πατήστε **«Έναρξη Μετατροπής»**
4. Το MP3 αποθηκεύεται στον ίδιο φάκελο με το αρχικό αρχείο
